# QuiclParking

## História dos Usuários

```
Registro de usuário, Sistema de Estacionamento:

Como usuário do estacionamento, eu quero poder me registrar no sistema para que eu possa usar os serviços de estacionamento de forma conveniente e eficiente.
```

```
Login do usuário, Sistema de Estacionamento:

Como usuário do estacionamento, eu quero fazer login com os dados do registro para que eu possa logar em diferentes dispositivos e usar o serviço de forma conveniente.
```

```
Registro de veículo e vaga, Sistema de Estacionamento:

Como usuário do estacionamento, eu quero realizar o cadastro do meu veículo com todas as informações dele, e registrar ele logo após em uma vaga do estacionamento, facilitando a reserva da vaga e tornando o serviço conveniente.
```

```
Checkout da vaga do estacionamento:

Como usuário do estacionamento, eu quero realizar o checkout do veículo usando a placa(Identificador), para que quando eu sair do estacionamento seja confirmada a saída do veículo na aplicação.
```
